# Data Professional Survey Dashboard

## Project Overview
Created an interactive Power BI dashboard analyzing survey responses from data professionals.

## Skills Used
- Power BI
- Power Query
- Data Cleaning
- Data Modeling
- DAX
- Interactive Dashboards

## Dashboard Features
- Average Salary by Job Title
- Country Distribution
- Favorite Programming Language
- Work/Life Balance
- Salary Satisfaction
- Difficulty Breaking Into Data
